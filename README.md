# Calculateur IP - IPv4 & IPv6

Un calculateur de sous-réseaux IPv4 et IPv6 avec des explications étape par étape en français.

## Fonctionnalités

- **Calculateur de Sous-réseau IPv4** : Calcul détaillé des sous-réseaux avec masques
- **Calculateur Notation Préfixe IPv4** : Calcul à partir de la notation préfixe (/24, /28, etc.)
- **Calculateur IPv6** : Calcul des réseaux IPv6 avec préfixes

## Explications étape par étape

Chaque calcul est décomposé en étapes détaillées :
- Méthode du bloc pour IPv4
- Conversion préfixe vers masque
- Calcul des bits d'hôte et de sous-réseau
- Détermination des adresses réseau, broadcast, première et dernière

## Technologies

- React 18
- Vite
- React Router
- CSS moderne avec thème clair

## Développement

```bash
npm install
npm run dev
```

## Déploiement

Le projet est configuré pour Vercel avec support du routing côté client.
